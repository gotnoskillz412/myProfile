Welcome to Your New Goals Headquarters!

Goal Tending is a place to tend to your goals. If you're like me, you have an easy time thinking of goals for yourself, but a hard time keeping track of them. I made this site to both improve my development skills, but also as a place I could keep track of the things I set out to do. This is a free service to anybody who wants to use it, so feel free to create a profile and use this application as you please!
This application is currently in its MVP stage, which stands for Minimum Viable Product. This application was released with the essentials to make it functional and useful. I plan to add more and more features, as well as add additional testing done on this page to increase performance and reliability. If you come across a problem on the page, please feel free to email me via the contact page visible once you make a profile.
Here is a list of some of the new features I'd like to add on to this application...
  • Accomplishment badges for completing your goals
  • Charts to keep track of how many goals you are completing and how long it usually takes
  • Filtering on the goals list page so you can easily get to the goal you are looking for
  • Sorting on the goals list page to make it that much easier
  • Due dates for each goal
  • Email notifications
Goal tending was created using the MEAN stack (MongoDB, Express framework, Angular, and Node). I used Bootstrap for the styling, and Angular CLI to get the app started. The backend API lives in a separate repository. If you are interested in seeing the source code for this application, you can see it here!
